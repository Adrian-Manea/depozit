Elementele formularelor HTML

Tag Nume Descriere
<form> Form Folosit pentru crearea de formulare ce colecteaza date de la utilizator
<input> Input Este cel mai utilizat element al unui formular. Poate fi redat in mai multe feluri, in functie de atributul type.
<label> Label Defineste o eticheta(label) pentru multe din elementele unui formular
<select> Select Defineste o lista drop-down
<textarea> Textarea Defineste un camp de text cu mai multe linii
<button> Button Defineste un buton pe care se poate apasa
<fieldset> Fieldset Este folosit pentru a grupa date similare intr-un formular
